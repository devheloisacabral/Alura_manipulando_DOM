# Alura_manipulando_DOM

Projeto realizado em um dos cursos da formação JS para frontend Alura. A maioria dos temas abordados neste curso ja eram de meu conhecimento por projetos anteriores, mas alguns itens novos foram apresentados, estes são:

Data attribute - atribuição de valores extras no HTML, com a finalidade de acessá-lo no JS sem que haja comprometimento do script caso alguma mudança no design seja feita.
ForEach - utilizado para otimização do código, diminui as linhas repetidas e atribui certa ação em cada elemento da ordem.
